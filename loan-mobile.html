Loan Computation Mobile App

Overview

The Loan Computation Mobile App was developed using Sketchware for Android to help loan officers perform computations during field visits and client meetings.

Problem It Solves

When visiting prospective clients, loan officers often needed to manually compute loan proceeds and monthly amortization because a desktop or laptop was not available.

This process required additional time and made it harder to explain loan products effectively.

The mobile application allowed instant loan computations directly from an Android device, making client discussions more efficient and professional.

Features

  • Mobile loan computation
  • Loan proceeds calculation
  • Monthly amortization calculation
  • Built using Sketchware
  • Field-use accessibility
  • Faster client consultations
  • Improved product presentation
